Arten von Buchlisten

  • globale Bücherliste
  • User können recherchieren und gefundene Bücher als BibTeX exportieren
  • keine lokalen Listen für User

Einteilung von Benutzern

  • Anonyme Nutzer (Gäste) können Suchen und Exportieren
  • Registrierte Nutzer können zusätzlich Bücher hinzufügen/ändern/löschen/kommentieren
  • Administratoren können zusätzlich Nutzer hinzufügen/ändern/löschen

Sichtbarkeit von Kommentaren

  • registrierte Nutzer können Kommentare zu Buch verfassen
  • jeder kann Kommentare zu Buch lesen

Funktionsweise der Suche

  • Suche nach spezifischen Informationen (Titel, Autor, Verlag, ...)
  • Freie Suche (Volltextsuche) nach Stichwörtern (auch in Bemerkungen, Stichworte, ...)

Informationen für Bücher

  • Informationen laut Aufgabenbeschreibung:
    • Art
    • Titel
    • Autor
    • ISBN
    • Jahr
    • Ort
    • Beschreibung
  • weitere Informationen laut DIN 1505 Teil 2
